# Dealer Onboarding

## Goal

Identify the dealer's real inventory/pricing source and configure LotSync correctly.

## Discovery Questions

1. What DMS do you use?
2. What inventory merchandising system do you use?
3. What website provider do you use?
4. Where does website pricing come from?
5. Is there an inventory feed?
6. Is the feed CSV, XML, JSON, API, FTP, or SFTP?
7. How often does it update?
8. Which price should be shown on the lot?
9. Are rebates/incentives included?
10. What disclaimers must appear?
11. Who can approve tag updates?
12. How many vehicles are on the lot?
13. How many tags are needed?
14. Where will gateways be installed?

## Setup Steps

1. Create dealership account.
2. Add users.
3. Configure inventory source.
4. Import vehicles.
5. Register ESL devices.
6. Install gateway.
7. Pair first test vehicle.
8. Verify price against website.
9. Run pilot batch.
10. Train lot staff.

## Training

Lot staff must learn:
- scan VIN
- scan ESL
- confirm pairing
- reassign tag
- report broken tag
- handle sold vehicle
