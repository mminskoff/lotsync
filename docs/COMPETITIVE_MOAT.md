# Competitive Moat

## What LotSync Is Not

LotSync is not just:
- ESL hardware
- a digital sticker
- a Minew dashboard
- a dealer website

## What LotSync Is

LotSync is the real-time synchronization layer for dealership physical inventory displays.

## Moat Areas

1. Dealer inventory normalization
2. VIN-to-ESL assignment workflow
3. Real-time sync engine
4. Price mismatch detection
5. Audit/compliance history
6. Hardware provider abstraction
7. Dealer onboarding playbook
8. Multi-rooftop operations

## Strategic Insight

Hardware can be copied.

Reliable dealer inventory synchronization and operational workflow are much harder to copy.
