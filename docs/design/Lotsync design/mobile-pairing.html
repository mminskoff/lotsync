The lot attendant flow

Ten screens engineered for one-handed use in bright sun while walking the lot. Oversized targets, a single dominant action per screen, mono identifiers for at-a-glance verification, and a happy path that completes a pair in under ten seconds — plus every state that can interrupt it.
