# Patent and IP Strategy

## Important

This is not legal advice. Speak with a patent attorney before public demos, investor decks, or dealer pilots.

## Potentially Protectable Areas

The broad idea of electronic shelf labels is not new.

Potentially protectable dealership-specific workflows may include:

- Real-time synchronization between dealer inventory systems and vehicle-mounted ESLs
- VIN-to-ESL pairing through scanning workflow
- Price mismatch detection between feed, website, and physical tag
- Dealer-lot reassignment workflow
- Audit trail proving displayed physical price
- White-labeled hardware abstraction for dealer ESL deployments
- Sync status/compliance workflow for vehicles

## Recommended Action

Before public launch:
1. Document invention clearly.
2. Record diagrams and workflows.
3. Speak with patent attorney.
4. Consider filing provisional patent.
5. Use NDA for early partner conversations when possible.
6. Protect LotSync name with trademark search/filing.

## Trade Secrets

Keep these private:
- Provider integration implementation
- Inventory normalization rules
- Dealer onboarding playbook
- Sync error handling logic
- Pricing mismatch heuristics
