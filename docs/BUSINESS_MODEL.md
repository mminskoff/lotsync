# Business Model

## Revenue Streams

1. SaaS subscription
2. White-labeled hardware kits
3. Setup/onboarding fees
4. Integration fees
5. Support/maintenance
6. Enterprise multi-rooftop contracts

## Example Packages

### Starter

- 1 gateway
- 100 ESL tags
- Dealer dashboard
- Mobile pairing app
- Basic inventory feed
- Monthly SaaS

### Professional

- 250 ESL tags
- Gateway support
- Advanced sync monitoring
- Price mismatch alerts
- Custom label templates

### Enterprise

- 500+ tags
- Multi-rooftop support
- Custom integrations
- SLA support
- Dedicated onboarding

## Pricing Variables

- Number of rooftops
- Number of active vehicles
- Number of ESL tags
- Integration complexity
- Support level
