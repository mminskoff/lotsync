# Pilot Deployment Plan

## Goal

Run LotSync with one dealership before scaling.

## Recommended Pilot Scope

- 25-50 vehicles
- 25-50 ESL tags
- 1 gateway
- 1 inventory source
- 2-5 dealer users

## Pilot Success Metrics

- 95%+ successful ESL updates
- Price updates under target SLA after LotSync receives data
- Zero cross-vehicle price errors
- Lot staff can pair tags without engineering help
- Dashboard clearly shows mismatches/failures
- Dealer confirms physical tags match inventory/website

## Pilot Phases

### Phase 1: Driveway Test

Use personal cars and Minew kit.

### Phase 2: Dealer Lab Test

Use a few vehicles at dealership.

### Phase 3: Small Lot Pilot

Use 25-50 vehicles.

### Phase 4: Full Lot Rollout

Expand to 100-400 tags.

## Pre-Pilot Checklist

- Provisional patent discussion completed
- Dealer NDA signed if needed
- Inventory source identified
- Hardware installed
- Mobile app pairing tested
- Audit logs working
- Rollback plan ready
